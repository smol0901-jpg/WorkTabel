# Архитектура WorkTabel

## Основные сущности:
- **Компания**
- **Точка (филиал)**
- **Сотрудник**
- **Роль** (админ, менеджер, оператор, аналитик, планировщик)
- **График работы**
- **Выручка**
- **ФОТ**
- **Внутренний чат** (чаты по ролям, сообщения, внутренние задачи)
- **Интеграция с Telegram** (уведомления, связь с каналами)
- **Интеграция с нейросетями** (GigaChat, ChatGPT и др.)

## ER-диаграмма (словесно):
- Компания —< Точка —< Сотрудник —< График работы
- Сотрудник —< Роль
- Точка —< Выручка
- Сотрудник —< Сообщение (чат)
- Внутренняя задача —< Сообщение —< Роль
- Чат может быть связан с Telegram-каналом

## Описание ролей и прав
- **Админ**: полный доступ, управление всеми сущностями
- **Менеджер**: управление точками, графиками, задачами
- **Оператор**: просмотр и отметки по графику, отчёты
- **Аналитик**: доступ к аналитике, выгрузкам
- **Планировщик**: создание и редактирование графиков

## Логика внутреннего чата
- Создание задач и сообщений по ролям
- Возможность обсуждений внутри приложения
- Привязка к Telegram-каналам (опционально)
- Использование для объявлений (инвентаризация, уборка и т.д.)

## Экспорт/импорт
- Выгрузка аналитики и графиков в HTML и PDF
- Импорт графика работы по шаблону (Excel/CSV)

## Интеграция с нейросетями
- Подключение GigaChat, ChatGPT для планирования, генерации идей, автоматизации
