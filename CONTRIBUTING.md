# Как участвовать в проекте WorkTabel

1. Ознакомьтесь с README и docs/.
2. Все предложения и баги — через Issues.
3. Для доработок — создайте отдельную ветку и Pull Request.
4. Следите за чек-листами и этапами.

---

## Как проверить ход работы
- Смотрите раздел Issues — там задачи и прогресс.
- После каждого этапа будет отчёт и инструкция по проверке.
