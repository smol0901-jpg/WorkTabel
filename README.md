# WorkTabel

**WorkTabel** — онлайн и десктоп-приложение для автоматизации графиков работы, учёта выручки, расчёта ФОТ и аналитики в малом бизнесе (общепит и не только).

## Основные возможности
- Создание и планирование графиков работы
- Учёт ежедневной выручки по точкам
- Автоматический расчёт ФОТ
- Дашборды и аналитика
- Многоуровневые роли пользователей (админ, оператор, аналитик, планировщик, менеджер)
- Работа с несколькими компаниями и точками
- Интеграция с Telegram для уведомлений
- Прогнозирование и планирование
